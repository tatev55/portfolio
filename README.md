# Portfolio Starter (React + Ant Design + Vite)

Tech: React, TypeScript, Vite, Ant Design, pnpm.

## Quick start

```bash
pnpm install
pnpm dev
```

## Where to edit your info

- Update your name, links, skills, and projects in `src/content/profile.ts`.
- Pages live in `src/pages/*`.
- Layout/nav/footer live in `src/app/layout/AppLayout.tsx`.

## Build

```bash
pnpm build
pnpm preview
```
